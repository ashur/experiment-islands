# eleventy-template

A simple [Eleventy](https://11ty.dev) project template.

## Installation

### Requirements

- [Node and NPM](https://nodejs.org)

### Steps

1. Clone this repo
1. Run `npm install`

## Usage

To build HTML and CSS from source, run:

```
npm run build
```

To watch changes to the site contents and run a small local web server, run:

```
npm run serve
```
